TCNJ BookNav

An ongoing research project I am leading at TCNJ to develop an indoor navigation system for the campus library. I have taken advantage of the access to WiFi and accelerometer data that the Android platform allows to develop a fingerprinting system that is easy to implement and has an average error of initial placement of 9.5 feet, which is competitive among current indoor navigation systems.
Information about this project is availible here: https://github.com/derricp1/navispace

Other Highlights
